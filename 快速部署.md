# ⚡ 快速部署指南

## 🎯 最快部署方式

### 步骤1：提交并推送代码
```bash
git commit -m "优化：全局性能优化，达到200+ FPS"
git push origin main
```

### 步骤2：在 Vercel 部署

#### 如果是第一次部署：
1. 访问 https://vercel.com/new
2. 使用 GitHub 登录
3. 导入仓库 `qyChiba/OneBk`
4. Vercel 会自动检测 Next.js 配置
5. 点击 **Deploy** 按钮
6. 等待构建完成（约2-3分钟）

#### 如果已经部署过：
- 推送代码后，Vercel 会自动触发部署
- 在 Vercel Dashboard 查看部署状态

---

## 🚀 使用 Vercel CLI 部署（可选）

### 安装 CLI
```bash
npm install -g vercel
```

### 登录
```bash
vercel login
```

### 部署
```bash
# 预览部署
vercel

# 生产环境部署
vercel --prod
```

---

## ✅ 部署后检查

1. **访问部署 URL**
   - Vercel 会提供：`https://your-project.vercel.app`

2. **验证功能**
   - ✅ 页面正常加载
   - ✅ 菜单栏功能正常
   - ✅ 每日一言正常显示
   - ✅ 动画流畅（200+ FPS）

3. **性能测试**
   - 打开 Chrome DevTools
   - Performance 面板测试 FPS
   - 应该达到 200+ FPS
