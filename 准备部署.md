```bash
部署优化：

```

### 第二步：推送到 GitHub（您来执行）

```bash
# 如果是新仓库（第一次推送）
git remote add origin https://github.com/你的用户名/OneBk.git
git branch -M main
git push -u origin main

# 如果已有仓库（后续推送）
git push
```

### 第三步：在 Vercel 部署

**方式一：网页部署（推荐，超级简单）**

1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "Add New..." → "Project"
4. 选择 "Import Git Repository"
5. 找到 `OneBk` 仓库，点击 "Import"
6. 保持默认配置，点击 "Deploy"
7. 等待 2-3 分钟 ⏱️
8. 🎉 部署成功！

**方式二：CLI 部署（可选）**

```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录
vercel login

# 部署到生产环境
vercel --prod
```
